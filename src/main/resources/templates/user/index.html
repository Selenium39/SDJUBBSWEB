上海电机学院（Shanghai DianJi University）是一所面向先进制造业及现代服务业，以工学为主，经济学、管理学、文学、艺术学等学科协调发展的的普通高等院校。被国务院学位办列为“服务国家特殊需求人才培养项目”专业学位研究生试点单位，入选教育部“卓越工程师教育培养计划”高校、国家级特色专业建设点、国家级新工科研究与实践项目、上海市首批深化创新创业教育改革示范高校、上海高等学校一流本科建设引领计划、上海高等学校一流研究生教育引领计划、全国职业教育先进单位、上海市促进就业先进集体、上海市文明单位，CDIO工程教育联盟成员。
